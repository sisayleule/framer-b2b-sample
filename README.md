# Nexora Solutions - B2B Landing Page

A modern, one-page B2B website built with React, Framer Motion, and Vite. Features a clean, minimal, enterprise-friendly design optimized for business clients.

- **Clean & Professional**: Minimal design with purple/blue gradient accents
- **Desktop-First**: Optimized for desktop viewing with full responsive support
- **Smooth Animations**: Subtle Framer Motion animations for enhanced UX
- **Enterprise-Ready**: Professional styling suitable for SaaS, consulting, and tech services

## üìã Sections

1. **Hero** - Full-width two-column layout with animated visual elements
2. **Services** - 3-column grid showcasing core offerings
3. **Why Choose Us** - Two-column layout with checkmark list
4. **How It Works** - 3-step process with numbered cards
5. **Testimonials** - Client feedback with sample badges
6. **CTA** - Full-width call-to-action with gradient background
